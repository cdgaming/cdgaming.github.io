<< 起動方法 >>
 Game.exeを実行する事でゲームがご利用頂けます。
　セキュリティーソフト等の設定が必要であれば許可して下さい。

<< アンインストール方法 >>
 このファイルがあるフォルダごと全て削除する事で可能です。
 セーブデータ含め全データはこのフォルダ内にあります。

　これらは1.04aでのクラウド化により必要なくなりました。
